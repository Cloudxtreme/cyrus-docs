===============
``cvt_cyrusdb``
===============

intro

Synopsis
========

.. parsed-literal::

    cvt_cyrusdb [OPTIONS]

Description
===========

The ``cvt_cyrusdb`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.
