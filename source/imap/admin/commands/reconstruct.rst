===============
``reconstruct``
===============

intro

Synopsis
========

.. parsed-literal::

    reconstruct [OPTIONS]

Description
===========

The ``reconstruct`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

=======
