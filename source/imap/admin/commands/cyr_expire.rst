==============
``cyr_expire``
==============

intro

Synopsis
========

.. parsed-literal::

    cyr_expire [OPTIONS]

Description
===========

The ``cyr_expire`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

=======
