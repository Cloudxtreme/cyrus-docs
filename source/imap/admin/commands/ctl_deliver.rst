===============
``ctl_deliver``
===============

Synopsis
========

.. parsed-literal::

    ctl_deliver [OPTIONS]

Description
===========

The ``ctl_deliver`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

Options
=======
