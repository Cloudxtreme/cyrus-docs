==============
``cyr_dbtool``
==============

intro

Synopsis
========

.. parsed-literal::

    cyr_dbtool [OPTIONS]

Description
===========

The ``cyr_dbtool`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

=======
