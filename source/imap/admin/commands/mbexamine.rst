=============
``mbexamine``
=============

intro

Synopsis
========

.. parsed-literal::

    mbexamine [OPTIONS]

Description
===========

The ``mbexamine`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

=======
