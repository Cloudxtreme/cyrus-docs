================
``ctl_mboxlist``
================

intro

Synopsis
========

.. parsed-literal::

    ctl_mboxlist [OPTIONS]

Description
===========

The ``ctl_mboxlist`` program outputs a list of files and/or directories
that it expects to exist, but that in fact do not.

=======
