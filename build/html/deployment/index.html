Deployment Guide¶

A Guide to Planning a Cyrus IMAP Deployment¶

A number of considerations need to be made prior to deploying Cyrus IMAP in an existing infrastructure, especially larger ones. This guide addresses these considerations, to aid in planning the successful deployment of Cyrus IMAP.
